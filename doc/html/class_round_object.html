float  endAngle = M_PI_2 
) [inline]

Constructs a RoundObject with the given material. The object is represented by an arc which goes from startAngle to endAngle. If endAngle > startAngle, it goes counter-clockwise from startAngle to endAngle; otherwise, it goes clockwise.

Parameters:
materialThe material for the object being created.
centerThe center of the circle for the moving object.
radiusThe radius of the circle for the moving object.
startAngleThe starting angle for the arc.
endAngleThe ending angle for the arc.
